## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- A local web server (optional, but recommended)

### Installation

1. **Clone or download this repository**
   ```bash
   cd brickell-air
   ```

2. **Open in browser**
   - Simply double-click `index.html` to open in your browser
   - Or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js (http-server)
   npx http-server
   
   # Using PHP
   php -S localhost:8000
   ```

3. **Add video (optional)**
   - Place a video file named `duct-cleaning-video.mp4` in the `videos/` folder
   - The video should be muted and loopable for best results

## 🎨 Customization

### Colors
Edit the CSS variables in `css/styles.css`:
```css
:root {
    --primary-color: #0066cc;      /* Main brand color */
    --primary-dark: #004c99;       /* Darker shade */
    --secondary-color: #00b3e6;    /* Secondary color */
    --accent-color: #ff6b35;       /* Accent color */
}
```

### Translations
Edit the `translations` object in `js/main.js` to modify texts in English and Spanish.

## 📱 WhatsApp Integration

The WhatsApp button sends a pre-filled message:
```
"Hello, I would like to inquire about your air duct cleaning services."
```

To customize, edit the message in `js/main.js` or the href in each HTML file.

## 🗺️ Google Maps

The contact page includes an embedded Google Map. To update the location:

1. Go to [Google Maps](https://www.google.com/maps)
2. Find your location
3. Click "Share" → "Embed a map"
4. Copy the iframe code
5. Replace the iframe in `contact.html`
